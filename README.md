# Immersive Calibration Framework

MSc Robotics dissertation repository for a VR assisted digital twin calibration framework for a 3D printed 6-axis robotic arm called ARCTOS.

## Current Contents

- `software/arctos_ws` — ROS 2 / Gazebo workspace
- `software/arctos_ros1_ws` — ROS 1 workspace / legacy reference

## Research Direction

This project investigates:

- ROS 2 robotic control
- Gazebo digital twin simulation
- Self-calibration pipelines
- Vision-assisted pose estimation
- Digital twin synchronization
- Optional VR integration
- Optional P2PRC remote telemetry and monitoring

## Goal

This is to Develop a distributed digital twin assisted calibration framework for low cost 3D printed robotic manipulators.
